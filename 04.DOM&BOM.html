DOM
문서 객체 모델(The Document Object Model, 이하 DOM) 은 HTML, XML 문서의 프로그래밍 interface 이다.
DOM은 문서의 구조화된 표현(structured representation)을 제공하며 프로그래밍 언어가 DOM 구조에 접근할 수 있는
방법을 제공하여 그들이 문서 구조, 스타일, 내용 등을 변경할 수 있게 돕는다. DOM 은 구조화된 nodes와 property 와 method
를 갖고 있는 objects로 문서를 표현한다. 이들은 웹 페이지를 스크립트 또는 프로그래밍 언어들에서 사용될 수 있게 연결시켜주는 역할을 담당한다.

javascript selector

구분 getElementsBy* querySelector*
id document.getElementById(‘foo’) (모든브라우저지원) document.querySelector(‘#foo’)
tag document.getElementsByTagName(‘li’) (모든브라우저지원) document.querySelectorAll(‘li’)
class document.getElementsByClassName(‘bar’) (ie9 이상지원) document.querySelector(‘.bar’)
속성(name=foo) document.getElementsByName(‘foo’) (모든브라우저지원) document.querySelectorAll(‘[name=foo]’)
querySelector = 하나의요소 선택시 사용(ie8 이상지원) , querySelectorAll = 여러개의요소 선택시 사용(ie9 이상지원) 
                            
//요소의 id 속성값으로 선택
var elem = document.getElementById('아이디명');


//요소의 class 속성값으로 선택
var elem = document.getElementsByClassName('클래스명');

//요소의 tagName 속성값으로 선택
var elem = document.getElementsByTagName('태그명');

//요소의 name 속성값으로 선택
var elem = document.getElementsByName('이름명');


//querySelector
document.querySelector('#foo'); //id 선택
document.querySelector('.foo'); //class 선택
document.querySelector('li'); //tag 선택
document.querySelector('[name=foo]'); //name 선택

//querySelectorAll
document.querySelectorAll('.foo'); //class 선택
document.querySelectorAll('li'); //tag 선택
document.querySelectorAll('[name=foo]'); //name 선택

//복수요소 선택시 활용방법
var elems = document.querySelectorAll('클래스명');
//or
var elem = document.getElementsByClassName('클래스명');

for (var i=0; i < elems.length; i++){
    elems[i].style.backgroundColor = 'blue';
}
//elems[인덱스값] 선택된 요소를 인덱스값으로 검색하여 사용가능

//객체안의 태그 검색
var idx = document.getElementById('test');
var x = idx.getElementsByTagName("LI");
for (var i=0; i < x.length; i++){
    console.log(x[i].innerHTML);
}

DOM event

브라우저는 이벤트를 감지할 수 있으며 이벤트 발생 시에는 통지해 준다. 이 과정을 통해 사용자와 웹페이지는 상호작용(Interaction)이 가능하게 된다.

이벤트 등록방법
In HTML: <element onclick="myScript">
In JavaScript: object.onclick = function(){myScript};
addEventListener() method: object.addEventListener("click", myScript); (ie9 이상지원)
event 참고 링크 
                            
//HTML 방식
"<button onclick="myFunction()">Click me</button>"


//JavaScript
document.getElementById("demo").onclick = function() {myFunction()};
function myFunction() {
    console.log('click');
}


//addEventListener 방식
document.getElementById("demo").addEventListener("click", myFunction);
function myFunction() {
    console.log('click');
}

자바스크립트를 사용해서 CSS를 조작하고 요소에 사용된 스타일을 동적으로 변경할수있다.
css 변경시 지원브라우저 확인필요

style 참고 링크 
                            
//객체 선택후 style 적용

//배경색 변경

elem.style.left = "100px";

BOM
BOM(Browser Object Model)이란 웹브라우저의 창이나 프래임을 추상화해서 프로그래밍적으로 제어할 수 있도록 제공하는 수단이다.
BOM은 전역객체인 Window의 프로퍼티와 메소드들을 통해서 제어할 수 있다.

//setInterval() 1초간격으로 화면상에 현재 시간을표시
var myVar = setInterval(function(){ myTimer() }, 1000);

function myTimer() {
    var d = new Date();
    var t = d.toLocaleTimeString();
    document.getElementById("demo").innerHTML = t;
}

function myStopFunction() {
    clearInterval(myVar);//setInterval 정지
}

//setTimeout() 3초후 Hello text를 경고창으로 표시
var myVar;

function myFunction() {
    myVar = setTimeout(function(){ alert("Hello"); }, 3000);
}

function myStopFunction() {
    clearTimeout(myVar);
}
